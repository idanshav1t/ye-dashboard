# Ye Dashboard 🖖

Control panel for Ye - your AI assistant.

## Features

- 📊 Status Overview
- 🧠 Skills Management
- 📈 Usage Statistics  
- 🔍 Memory Search
- ⚡ Quick Actions
- 📁 File Browser
- ⚙️ Configuration
- 📜 Activity Logs

## Access

- **Web:** https://ye-dashboard.vercel.app
- **Mobile:** Responsive design for iPhone access

## Tech Stack

- Next.js
- Vercel

## Usage

This dashboard connects to the OpenClaw VPS to display and control Ye.
